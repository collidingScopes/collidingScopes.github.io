About

This web tool is completely free, open source, without any paywalls or premium options. You are welcome to use it for personal or commercial purposes.

If you found this tool useful, feel free to buy me a coffee. This would be much appreciated during late-night coding sessions!

Enormous thanks to Luke Hannam, whose blog post explained the code and mechanics for creating kaleidoscope animations in javascript.

I made only a few tweaks to Luke's original algorithm, and then added the front-end user interface which allowed users to upload their own photos, and to control variables like animation speed.

A few tips for using this tool:

  • Press "p" to pause / play the animation. This lets you stop at an interesting point of the animation
  • Press "s" to save a screenshot of the current state of the animation
  • To save the animation as a video or gif file, I suggest using free tools like OBS studio or ScreenToGif. Both of those tools let you record a portion of your screen
